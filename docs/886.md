# SVG polyline element

> 原文:[https://jenkov.com/tutorials/svg/polyline-element.html](https://jenkov.com/tutorials/svg/polyline-element.html)

SVG `<polyline>`元素用于绘制多条相连的线(poly = multiple)。

## SVG 折线-视频教程
