# SVG 椭圆元素

> 原文：<https://jenkov.com/tutorials/svg/ellipse-element.html>

SVG `<ellipse>`元素用于绘制椭圆。椭圆是一个高度和宽度不等的圆。换句话说，它在 x 和 y 方向的半径是不同的。
